# Micro-Blogging-App MERN Stack
## Features
1. Create account and Login
2. Write posts(micro-blogs)
3. Like and Unlike posts
4. Follow and Unfollow other users
5. View posts of users you follow

## Live Demo
https://socio-page.herokuapp.com/
